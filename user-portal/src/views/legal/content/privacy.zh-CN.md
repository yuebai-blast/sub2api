本政策适用于 Mintpop 提供的 API 中转、额度管理、订阅、兑换码、调用统计、账户管理、风控与安全等相关技术服务。

本平台主要面向海外华人、海外开发者、跨境团队等合法合规的境外用户。

### 二、我们可能收集的信息

为提供、维护、改进和保护我们的服务，我们可能根据功能需要收集以下信息：

- **账户信息：** 用户名、邮箱地址、密码摘要、头像、角色、用户组、注册时间、登录状态、账户状态、邀请码或邀请关系。
- **安全与登录信息：** IP 地址、登录时间、设备与浏览器信息、User-Agent、会话标识、登录失败记录、双因素认证状态、异常访问记录。
- **交易与额度信息：** 套餐、订阅、兑换码、充值记录、额度余额、额度变动、订单状态、支付渠道返回的必要交易信息、发票或账务信息。
- **API 使用信息：** API Key 标识、调用时间、模型名称、请求路径、消耗额度、响应状态、错误信息、用量统计、限流记录、并发记录、渠道或分组信息。
- **用户提交的信息：** 工单、客服沟通内容、反馈、申诉材料、合规说明、备注，或您主动上传或提交的其他内容。
- **Cookie 与本地存储：** 用于维持登录状态、保存偏好设置、识别安全风险、改善页面体验、避免重复提示的必要信息。

我们不会主动要求您提交与服务无关的敏感个人信息。请勿在工单、备注、请求内容或其他输入项中提交身份证件、银行卡密码、生物识别信息、健康信息、未成年人信息或其他不必要的敏感信息。
